Dinner Genius App

The motivation behind this app was helping to find dinner recipes based on ingredients the user already has on hand. A specific calorie range can be added as a filter. Ten results are displayed along with a suggested wine-pairing. The app uses the Fetch API to make an AJAX request to multiple APIs (Edeman and Spoonacular). Tech stack includes: HTML, CSS, Javascript, and jQuery.
